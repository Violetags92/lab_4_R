This package contains the function linreg that creates a linear regression using ordinary least squares. This function takes a formula and a data.frame and returns and linreg object.

library(mypackage)

For example, we take the iris data.frame and the formula PetalLength~Species, and create our linreg object:

lin1 <- linreg(formula = Petal.Length ~ Species, data = iris)

The linreg function contains differents methods implemented:

print(): we should have the printout as call of the regressions coefficients.

lin1$print()
## Call:
## lm(formula = Petal.Length ~ Species ,data=iris) 
## 
## Coefficients:
##              (Intercept) Speciesversicolor Speciesvirginica
## Petal.Length       1.462             2.798             4.09
